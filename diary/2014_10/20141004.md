### あー、あったわー

結論から言うと、純粋関数型プログラミング言語であるHaskell[^3] で作られた、Pandoc[^4] ってのを使うと出来ました。  

```Bash
$ git checkout -b bugFix#1
```

```bat
@echo off
pandoc.exe
```

こんな風に `$ ../template/generate.bat 20141004 10月4日` インライン表示も可能！！
